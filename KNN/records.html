K-Nesrest Neighbors
     A supervised learning based on distance measurement
  •      advantage
               high accuracy,insensitive to outliers,no assumptions about data（精度高，对异常值不敏感，无数据输入假定）
  •      disadvantage
               computation expensive,requires a lot of  memory （计算复杂度高，消耗大量内存）

How to perform K-Nesrest Neighbors algorithm

  1. extract dataset and labels
    • dataset is the real data that is been classified accurately
    • labels represent the classification of every line of data
  2. calculate and normalize the distances (normalize : make distances between 0 to 1)
  3. count the classification of the nearest k neighbors and then choose the most one
  4. test the algorithm
          split the dataset to two part,one is used as the accurate dataset,another one is used to predicate
ex.
      a dataset of size 1000, you can split it to two dataset , the smaller one is 100,the bigger one is 900.Use each point in           smaller one to test the algorithm and count the error times.Finally,error count divide the size of the smaller one,here is 100,then you can get the rate of error.
